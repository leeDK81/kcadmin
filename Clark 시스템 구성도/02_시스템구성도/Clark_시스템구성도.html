절대 금지

🔐 CI 외부 전송 금지

CI(연계정보) 단방향 암호화 값은 아이지넷 내부 DB에서만 관리합니다. 더파트너스·GA사·원수사 등 외부로 절대 전송하지 않습니다.

목적: 탈퇴 후 재가입 시 동일인 확인 (중복 가입 방지)
보관: 탈퇴 후 1년 후 자동 파기

반드시 선택

🤖 LLM은 국내 서버 운영 필수

AI LLM이 해외 서버를 경유하면 개인정보 국외 이전 규제 적용. 아래 3가지 방안 중 반드시 1개 선택해야 합니다.

① AWS Bedrock (서울 리전) ← 1순위
② HyperCLOVA X (네이버) ← 2순위
③ Azure OpenAI (한국 중부) ← 3순위

절대 금지

🚫 마이데이터 + AI 직접 결합 금지

아이지넷이 수집한 마이데이터를 AI(LLM)에 직접 입력하는 구조 불가합니다.

신용정보법: 마이데이터는 동의 목적 범위 내에서만 활용

→ 아이지넷은 데이터 전달만, 더파트너스(GA사)가 AI 분석 담당으로 역할 분리 필수

동의 필수

💊 건강 데이터 별도 동의

건강 검진 정보는 민감정보(개인정보보호법 제23조)로 분류됩니다.

일반 개인정보 동의와 반드시 분리된 별도 동의 화면 필요.
더파트너스 전송 시 더파트너스 명시 필수.
미동의 시에도 기본 서비스(마이데이터) 이용 가능해야 함.

허용

✅ 더파트너스(GA사) AI 보험 분석

더파트너스는 GA사(법인보험대리점)로 보험업법상 보험모집 허가 주체입니다.

→ KC카드 기반 LLM 보험 보장 분석 적법
→ 아이지넷 혁신금융 검수 범위에서 AI 서비스 제외 가능
→ 단, 면책 고지 화면 표시 필수

운영 기준

🔄 상담 철회 시 DB 즉시 회수

사용자가 상담을 철회하면 GA사/원수사에 철회 API 즉시 호출, 삭제 완료 응답(timestamp) 수신 후 감사 로그에 기록해야 합니다.

SLA: 24시간 이내 삭제 완료
완료 후 사용자에게 삭제 완료 알림 발송

운영 기준

⚖️ GA 배분 알고리즘 공정성

더파트너스(자회사)가 GA 배분 대상이 될 수 있어 공정거래법상 이해충돌 이슈 주의.

→ 배분 알고리즘 객관적 기준 문서화
→ 더파트너스 배분 시 "자회사입니다" 고지 의무
→ 더파트너스 우선 배분 조항 없음을 명시

허용

✅ 원수사 직접 DB 제공

마이데이터 사업자는 사용자 동의 기반 제3자 제공이 합법입니다.

기존 플랫폼에서 이미 운영 중인 검증된 구조.
동의 화면에 원수사명·제공항목·보존기간 명시 필수.

가드레일 필수

🚦 채팅 AI — 범위 밖 질문 차단

채팅 UI에서 어떤 질문이든 입력될 수 있어 범위 밖 답변이 생성될 경우 보험업법·금소법 저촉 위험이 있습니다.

차단 대상: 상품 추천 / 보험 비교 / 보험료 예측 / 가입 권유
차단 시 응답: "전문 상담사와 직접 상담을 권장합니다"로 유도
허용 범위: 현재 보장 현황 분석·공백 안내에 한정

허용

👤 비회원 AI 채팅 — 팝업 동의로 허용

AI 채팅 이용을 위한 더파트너스 서비스 동의는 회원가입 없이 팝업 동의 1회로 처리 가능합니다.

동의 ≠ 회원가입 — 쿠키 동의와 동일한 구조.

비회원: 팝업 동의 → 일반 보험 정보 AI 채팅 (개인 데이터 미사용)
로그인 회원: AI 채팅 + 상담 신청 + 대화 이력
마이데이터 연결: 내 보험 기반 맞춤 진단 + AI 채팅

면책 고지 필수

💬 생성형 AI 답변 — 면책 고지 상시 표시

생성형 AI가 자유롭게 답변할 경우 잘못된 보험 정보 제공 시 금소법 설명의무 위반 가능성이 있습니다.

모든 AI 답변 하단 상시 표시:
"이 답변은 AI가 생성한 정보로 정확성을 보장하지 않습니다. 보험 가입 전 전문 설계사와 상담하시기 바랍니다."
