# 컴플라이언스 에이전트

## 페르소나
금융 규제 및 개인정보 보호 전문가. 마이데이터 사업자 요건, 관련 법규에 정통하다.

---

## ✅ 최종 확정 — 법적 판정 (2026-05-14)

### Clark 서비스 범위 (확정)

| 기능 | 운영 주체 | 법적 근거 |
|------|---------|---------|
| 소셜 로그인 / 본인인증 | 아이지넷 | 개인정보보호법 동의 |
| 보험 마이데이터 조회 | 아이지넷 | 신용정보법 (마이데이터 허가 범위) |
| 건강 데이터 연동 | 아이지넷 | 개인정보보호법 민감정보 별도 동의 |
| KC카드 기반 보장 현황 분석 AI | **더파트너스** | GA사 허가 범위 내 |
| 상담 신청 시 원수사/GA에 고객 DB 전달 | 아이지넷 | 기존 사업에서 검증된 구조 |

> **서비스 범위 외**: 상품 추천, 보험료 예측, 보험 비교 — Clark 서비스에 해당 없음

### 기관별 판정 결과

| 기관 | 항목 | 판정 | 근거 |
|------|------|------|------|
| 신정원 | 마이데이터 → 더파트너스 전달 | ✅ 해소 | 업무 위탁 구조 — 위탁 계약서로 처리 |
| 금감원 | GA 배분 행위 보험모집 해당 여부 | ✅ 해소 | 기존 앱에서 동일 구조 운영 중 (검증됨) |
| 혁금위 | 규제 충돌 여부 | ✅ 불필요 | 양사 각자 허가 범위 내 운영 |
| 금융보안원 | 보안 요건 | ✅ 충족 | mTLS·AES-256·망 분리·CI 미전송 반영 |
| 개인정보위 | 민감정보·CI·철회·국외이전 | ✅ 충족 | 별도 동의·철회 API·Bedrock 서울 리전 |
| 보험업법 | AI 분석의 보험모집 해당 여부 | ✅ 안전 (GA사 운영) | 더파트너스=GA사 → 보험업법상 허가 주체 |

### 핵심 해소 근거
1. **아이지넷 → 더파트너스 = 업무 위탁** (제3자 제공 아님)
   - 유권 해석 불필요, 위탁 계약서 + 처리방침 공개로 해결
2. **상담 신청 DB 전달 = 기존 사업에서 이미 검증된 구조**
   - 아이지넷 기존 플랫폼에서 동일하게 운영 중
