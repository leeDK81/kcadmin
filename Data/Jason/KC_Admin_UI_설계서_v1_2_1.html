카드 테이블 (라이브러리)
  • 컬럼: Card ID, 타입, 리스크 유형, 공개범위, 버전, 승인일
  • Card ID 클릭 시 카드 상세 조회 화면(슬라이드 패널)으로 이동
  • 타입은 컬러 배지로 구분 (CONCEPT=파랑, RULE=초록, EVIDENCE=연두 등)
  • 공개범위는 아이콘+텍스트로 표시 (🟢 Public / 🟡 Baseline / 🔴 Internal)
  • 정렬: 각 컬럼 헤더 클릭으로 정렬 전환 (승인일순 기본)
  • 페이지네이션 또는 무한 스크롤
H
필터 바 (내 카드)
  • 라이브러리 대비 필터가 강화됨: 타입 + 상태 + 공개범위 3중 필터
  • 상태 필터: DRAFT / IN_REVIEW / LEGAL_REVIEW / APPROVED / REJECTED
  • 상단에 "새 카드 생성" 버튼 표시 (담당자만)
  • 필터 조합 시 AND 조건으로 적용
I
카드 테이블 (내 카드)
  • 라이브러리 대비 "상태" 컬럼이 추가됨
  • 상태 배지: DRAFT(파랑), IN_REVIEW(노랑), LEGAL_REVIEW(핑크), APPROVED(초록), REJECTED(빨강)
  • DRAFT / REJECTED 상태: "편집" 링크 표시 → 카드 에디터로 이동
  • IN_REVIEW / APPROVED 상태: "상세" 링크 표시 → 카드 상세 조회로 이동
  • REJECTED 카드는 행 배경을 연한 빨강으로 강조 (옵션)
★
라이브러리 vs 내 카드 차이 요약
  • 라이브러리: APPROVED 카드만 표시, 읽기 전용, 유형 탭 필터 제공
  • 내 카드: 모든 상태의 카드 표시, 편집 가능, 상태 필터 추가 제공
  • 두 뷰는 사이드바 메뉴로 전환 (별도 페이지)
  • 검수자/준법감시인은 "내 카드"에서 자신이 검수한 카드도 조회 가능
