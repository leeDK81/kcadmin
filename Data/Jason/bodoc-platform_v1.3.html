영역 1. AI 보장 진단 플랫폼

신용정보법 기반 마이데이터 사업자로서 사용자의 보험 데이터를 수집·분석하고 AI 진단을 제공하는 핵심 서비스 영역

📱
Channel 01

Clark (B2C)

일반 사용자가 마이데이터 기반으로 보험 진단을 받고 AI 상담 후 실제 상담을 신청하는 소비자 서비스

인증 · 보안
  • 인증소셜 로그인 (카카오, 네이버) · 본인인증 · CI 단방향 암호화
  • 인증자동로그인 / 세션 관리 / 토큰 갱신 / 회원가입 / 회원탈퇴 / 개인정보 수정
  • 인증자동 로그인 · 페이스 아이디 · 간편 비밀번호 사용 및 관리
  • 약관필수·선택 약관 동의 / 최신 약관 재동의
  • 보안개인정보 암호화 · SSL Pinning · 루팅/탈옥 감지
마이데이터
  • 마이데이터연결 / 해제 / 데이터 갱신
  • 마이데이터마이데이터 서비스 가입현황 — 연결 기관 목록 · 연결 상태 · 마지막 갱신 일시 조회
보험 가입현황
  • 보험현황내 보험 가입내역 목록 — 마이데이터 연결 사용자 기준 보험 계약 목록 · 보험사 · 상품명 · 보험료 · 만기일 · 계약 상태 표시
  • 보험현황보험 가입내역 상세 — 계약 기본정보 (계약번호 · 계약일 · 만기일 · 보험료) · 보장 항목별 보장금액 · 피보험자 · 수익자 조회
건강 데이터 연결
  • 건강데이터메디에이지 연결 / 업데이트 / 연결 해제 — KC 카드 보장 진단 데이터 소스
상담
  • 상담상담 신청 / 철회 / 이력 조회
알림 · 마케팅
  • 알림앱 알림설정
  • 마케팅Deeplink · Amplitude · UTM · Attribution · 강제 업데이트
고객 지원
  • 지원공지사항 / 이용약관 열람
  • 고객센터고객센터
법적 준수 사항
  • 전송요구권 — 정보제공기관 · 수신기관 · 대상정보 직접 선택 화면
  • 전송요구 철회 — 철회 요청 즉시 처리 의무
  • 열람권 — 저장된 본인 개인신용정보 조회 기능
  • 제3자 제공 현황 열람 — 정보 제공처 사용자 확인
  • 동의 이력 불변 저장 · 동의 철회 시 수집 정보 즉시 삭제
  • 연령 제한 — 14세 미만 가입 차단 / 19세 미만 정보활용 제한
  • 미접속자 보호 — 6개월 전송 중단 / 1년 자동 삭제 안내
  • 제3자 정보 제공 — 금융보안원 안심 제공 시스템 경유 의무
  • 이익 제공 제한 — 가입 유도 목적 3만원 초과 제공 금지
핵심 기능 (차별화)

  • AI 진단Concept → Risk-type → Rule → Evidence → Policy 체이닝 기반 진단 결과 제공
  • AI 상담AI 챗 기반 보험 Q&A (KC 카드 참조 · Playbook 기반 리드 전환)
  • 독립 웹뷰Clark/Lois 독립 웹뷰로 배포 가능 — 단독 앱 또는 헬스케어·핀테크 파트너 임베디드
도입 솔루션
  • 본인인증PASS / KCB / NICE — 마이데이터 연동 필수 인증 수단
  • 통합인증마이데이터 표준 통합인증 연동 (1회 인증 → 다수 정보제공기관 접근)
  • DB 암호화개인정보 저장 암호화 솔루션 (개인정보보호법 · 신용정보법)
  • 취약점 점검금융보안원 API 적합성 심사 대응 — 서비스 오픈 전 필수
  • 감사 로그마이데이터 접속 이력 불변 보관 솔루션 (1년 이상)

Channel 03

CRM (GA / 원수사용)

  • 상태고객 상태 변경 (미연락 · 연락중 · 완료 · 실패)
  • 상담메모 · 태그 · 분류 / 담당자 배정
  • 상담상담 이력 타임라인
DB 배정 관리
  • 배정 완료배정 완료 DB 현황 조회
  • 미배정미배정 DB 목록 · 수동 배정
