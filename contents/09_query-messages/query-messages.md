## QM-01 | 전체 보유 목록

**정의:** 사용자가 가입한 보험 상품 전체를 한 번에 나열해달라는 요청.

**응답 템플릿(안):** "현재 마이데이터에 연동된 보험은 총 N건이에요" + 상품별 리스트(보험명 / 월납입액 · 가입일)

**Q(예시 발화):**
- 내가 가진 보험 다 보여줘
- 나 지금 가입한 보험 뭐뭐 있어?
- 가입한 보험 목록 좀 알려줘
- 내 보험 리스트 뽑아줘
- 지금 들고 있는 보험 종류 알려줘
- 나 보험 몇 개나 들었어?
- 마이데이터 연동한 보험 전부 보여줘
- 내가 가입한 보험사랑 상품명 다 알려줘
- 우리 집에 든 보험 몇 개인지 궁금해
- 내 명의로 된 보험 전체 조회하고 싶어
- 가입해둔 보험 싹 다 정리해서 보여줘
- 나 보험 뭐 들었는지 까먹었어, 다 알려줘
- 전체 가입 내역 확인하고 싶어요
- 내 보험 현황 한눈에 보고 싶어
- 보유 중인 보험 상품 목록 알려줘
- 지금까지 든 보험 다 나열해줘
- 내 앞으로 되어있는 보험 전부 조회해줘

## QM-02 | 특정 상품 가입 여부

**정의:** 특정 보험 상품 하나에 가입돼 있는지를 확인하는 요청. 상품명은 `contents/10_synonym/synonyms.md`의 유사어 연결을 거쳐 표준 상품유형으로 정규화된 뒤 마이데이터 조회에 쓰인다.

**응답 템플릿(안):** "네, OO에 가입되어 있으시네요" + 해당 상품 상세(보험명 / 월납입액 · 가입일) / "아니요, OO은 확인되지 않아요"

**Q(예시 발화 — 상품명은 유사어 연결 대상이라 여기서는 표준 상품유형명으로 대표 기재, 실제 사용자 발화는 유사어 그대로 들어옴):**
- 나 운전자보험 가입되어 있어?
- 실손보험 있나?
- 암보험 들었었나 확인해줘
- 나 치아보험 가입한 적 있어?
- 상해보험 들어놨나?
- 간병보험 있는지 궁금해
- 화재보험 가입돼 있어?
- 나 연금보험 든 거 있나?
- 어린이보험 가입했었나 기억이 안 나
- 종신보험 들어놨어?
- 실비보험 있는지 확인 좀
- 정기보험 가입한 적 있나요?
- 암보험 가입 여부 알려줘
- 나 운전자보험 있나 없나 궁금해
- 태아보험 들었었나?
- 치과보험 가입돼 있는지 봐줘

## QM-03 | 갱신 예정 조회

**정의:** 조만간 갱신 시점이 도래하는 보험이 있는지 확인하는 요청.

**응답 템플릿(안):** "N개월 내 갱신 예정인 보험이 N건 있어요" + 상품별 상세(보험명 / 갱신 예정일 · 현재 월납입액)

**Q(예시 발화):**
- 나 곧 갱신되는 보험 있어?
- 갱신 예정인 보험 알려줘
- 이번 달에 갱신되는 거 있나?
- 다음 갱신일이 언제야?
- 보험료 오르는 갱신 시기 다가온 거 있어?
- 갱신형 보험 중에 곧 만기 되는 거 있나
- 갱신 앞둔 보험 몇 개나 있어?
- 재계약 임박한 보험 알려줘
- 조만간 갱신되는 보험 뭐 있는지 궁금해
- 갱신 기간 얼마 안 남은 보험 있어?
- 다가오는 갱신일 알려줘
- 내 보험 중에 갱신 준비해야 하는 거 있나?
- 갱신되면 보험료 얼마나 오르는지 미리 알고 싶어

## QM-04 | 만료 건수 조회

**정의:** 이미 만료됐거나 해지된 보험이 몇 건인지 확인하는 요청.

**응답 템플릿(안):** "현재 만료된 보험은 N건이에요" + 상품별 상세(보험명 / 가입일 · 만료일)

**Q(예시 발화):**
- 나 만료된 보험은 몇 개야?
- 끝난 보험 있어?
- 해지된 보험 몇 건인지 궁금해
- 만기 지난 보험 알려줘
- 유지 안 되고 있는 보험 있나?
- 보장 끝난 보험 몇 개인지 확인하고 싶어
- 만료돼서 효력 없는 보험 있어?
- 지금 실효된 보험 몇 개야?
- 예전에 들었다가 끝난 보험 있나
- 계약 종료된 보험 몇 건이야?
- 더 이상 보장 안 되는 보험 몇 개인지 알려줘

## 신규 등록 유형 (개발 반영 대기)

### QM-05 | 계약대출 가능 여부/현황 조회 (2026-07-14 신규 — 개발 반영 대기)

**정의:** 본인이 가입한 보험 중 계약대출(약관대출)이 가능한 상품이 있는지, 또는 현재 실행 중인 계약대출 현황(대출원금·이자)이 있는지 확인하는 요청.

**등록 근거:** 마이데이터 보험 업권 정보제공 API 규격(developers.mydatakorea.org)에 "대출상품" 카테고리가 실제로 존재함을 확인 — 기존 4종과 동일하게 마이데이터 직접 조회로 구현 가능성이 높다고 판단해 등록. 단, 응답 로직(대출 가능액 계산 등) 자체는 개발팀 구현 전까지 "개발 반영 대기" 상태.

**응답 템플릿(안):** "OO보험에서 계약대출이 가능하시네요(대출 가능액 약 N원)" 또는 "현재 실행 중인 계약대출이 N건 있어요" + 상품별 상세(보험명 / 대출 가능액 또는 대출원금·이자)

**Q(예시 발화):**
- 나 보험으로 대출받을 수 있어?
- 약관대출 가능한 보험 있나?
- 내 보험 중에 계약대출 되는 거 있어?
- 지금 대출받고 있는 보험 있나?
- 보험계약대출 얼마까지 가능해?
- 보험 담보로 얼마 빌릴 수 있어?
- 내가 받은 약관대출 현황 알려줘
- 보험대출 이자 얼마나 붙었는지 궁금해
