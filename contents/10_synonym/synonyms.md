# 유사어 관리 콘텐츠 — 상품유형 유사어 연결

> `mockups_v2/21_synonym-management.html`(운영자 등록 UI)에 실제로 입력할 유사어 연결 콘텐츠 원본.
> **역할:** 사용자가 혼용하는 상품명·구어체 표현을 마이데이터 표준 상품유형에 **결정론적(1:1)**으로 연결한다. AI 유사도 매칭 대상이 아니다 — 정책 배경은 `context/answer-logic.md` "예시 발화·상품유형 유사어 등록 방식" 참조.
> **등록 원칙:** 완전히 같은 상품의 다른 이름일 때만 연결한다. 이름이 비슷해도 실제로 다른 상품(보장 내용이 다른 상품)은 절대 연결하지 않는다 — 아래 "등록 금지 예" 참조.
> **2026-07-07 신설**: 실사용을 가정한 유사어 1차 저작. 표준 상품유형은 `guides/insurance-domain.md`의 T코드·담보코드 체계에서 실제로 다루는 상품군을 기준으로 선정했다.
> **2026-07-07 2차 갱신 — `insane-research`로 실제 신조어·줄임말·혼용 사례 조사**: 연금보험↔연금저축보험이 세제 취급(세제비적격 vs 세제적격)이 근본적으로 달라 별개 상품임을 확인 → 신규 등록 금지 예로 추가. 실손의료보험의 "구실손"·"신실손"은 1차로 유사어 후보에 넣었으나, 출처가 금융사 설명 콘텐츠(KB·교보생명 등)의 세대 구분 명칭일 뿐 실제 커뮤니티에서 자연발생적으로 쓰이는 신조어라는 확증은 아니어서 제외했다.
> **2026-07-07 3차 갱신 — "다른 이름/별칭" 관점으로 재조사(1·2차는 "은어/줄임말" 검색이라 놓친 게 있었음)**: 치아보험에 "치아연금"(가입자들이 붙인 실제 별명), 어린이보험에 "어른이보험"(성인도 가입 가능해 붙은 별명)·"신생아보험"(어린이보험의 한 종류로 확인) 추가. 동시에 새로운 혼동 상품 3건도 확인해 등록 금지 예에 추가: 종신보험↔CI보험/GI보험(중대질병 특약이 더해진 별도 상품, 보험료 30~50% 비쌈, 선지급 방식), 간병보험↔치매보험(치매보험은 치매 진단에만 한정, 간병보험은 원인 무관 간병 필요 상태 전반 보장), 화재보험↔재산종합보험(재산종합보험은 화재 외 도난·풍수해까지 포괄하는 더 넓은 별도 상품). 암보험·운전자보험·상해보험·정기보험·연금보험은 추가 조사했으나 확인되는 새 표현 없음.
> **2026-07-07 4차 갱신 — 구어체 단어 대체 패턴 반영(사용자 지적)**: 사용자가 "주택화재보험도 '집화재보험'처럼 쓸 수 있지 않냐"고 지적 — 1~3차는 전부 "공식적으로 존재하는 다른 이름"을 웹 검색으로 찾는 방식이었는데, 이런 일상적 단어 대체(주택→집처럼 formal 단어를 일상어로 바꾸는 것)는 검색으로 안 잡히고 한국어 어휘 패턴으로 직접 도출해야 함을 확인. 이 방식으로 유사어를 추가하되, **다른 상품유형과 겹칠 수 있는 지나치게 일반적인 단어(예: "사고보험")는 제외** — 결정론적 연결의 정확성 원칙을 유지하기 위해 해당 상품유형에만 명확히 대응되는 표현만 반영했다.

## 등록 금지 예 — 이름은 비슷해도 실제로 다른 상품

운영자가 유사어를 등록할 때 아래처럼 "표현은 비슷하지만 실제 보장 내용이 다른 상품"을 절대 같은 표준 상품유형으로 묶지 않는다. 벡터 유사도 매칭이 아니라 결정론적 연결을 쓰는 이유가 바로 이 문제를 막기 위해서다(`context/answer-logic.md` 참조).

| 헷갈리기 쉬운 표현 | 실제로는 | 왜 다른가 |
|---|---|---|
| 자동차보험 | 운전자보험이 아님 | 자동차보험은 차량에 대한 배상책임 의무보험, 운전자보험은 운전 중 형사합의금·벌금·변호사비용 등을 보장하는 별도 임의보험. 서로 다른 법적 성격의 상품 |
| 요양보험 | 간병보험이 아님 | "요양보험"은 국민건강보험공단이 운영하는 공적 장기요양보험(사회보험)을 가리키는 경우가 많고, 간병보험은 민영 보험사가 판매하는 사보험. 소비자가 혼용해도 운영 주체·보장 방식이 전혀 다름 |
| 생명보험 | 특정 표준 상품유형이 아님 | "생명보험"은 종신보험·정기보험·연금보험 등을 포괄하는 상위 카테고리 용어라 어느 한 표준 상품유형에 연결할 수 없음. 상위 카테고리 용어는 유사어로 등록하지 않는다 |
| 종신보험 ↔ 정기보험 | 서로 유사어 아님 | 종신보험은 평생 보장, 정기보험은 일정 기간만 보장 — 보장 기간·보험료 구조가 다른 별개 상품이라 하나로 합치면 안 됨 |
| 태아보험 | 어린이보험과 동일 상품 아님 | 태아보험은 임신 중 가입해 출생 전후 위험을 보장하는 시점 한정 상품이며, 출생 후 어린이보험으로 별도 전환되는 구조. 관련은 있으나 동일 상품은 아님 |
| 연금보험 ↔ 연금저축보험 | 서로 유사어 아님 (2026-07-07 조사) | 연금보험은 세제비적격 상품(생명보험사 판매, 납입 시 세액공제 없음, 일정 요건 충족 시 보험차익 비과세)인 반면 연금저축보험은 세제적격 상품(연 1,800만원 한도 납입, 연 600만원까지 세액공제, 수령 시 연금소득세 부과)이라 세제 혜택 구조 자체가 다른 별개 상품. 사용자가 "연금보험 있어?"라고 물었을 때 연금저축보험만 있는 경우 혼동해 답하면 세제 관련 사실관계 오류로 이어질 수 있음 |
| 종신보험 ↔ CI보험/GI보험 | 서로 유사어 아님 (2026-07-07 조사 신규) | CI보험(중대질병보험)은 종신보험에 중대질병(중대한 암·뇌졸중·급성심근경색 등) 진단 시 사망보험금 50~80%를 미리 지급하는 특약을 더한 별도 상품이며, 종신보험보다 보험료가 30~50% 비싸다. GI보험은 CI보험에서 "중대한"이라는 제한을 뺀 변형 상품. 보장 조건·가격이 크게 달라 같은 상품으로 취급하면 안 됨 |
| 간병보험 ↔ 치매보험 | 서로 유사어 아님 (2026-07-07 조사 신규) | 치매보험은 치매(경도~중증) 진단 시에만 진단금이 지급되는 반면, 간병보험은 중풍·파킨슨병·암·외상 등 치매 여부와 무관하게 간병이 필요한 상태 전반을 보장한다. "치매간병보험"처럼 결합 상품도 있어 헷갈리기 쉽지만 순수 치매보험과 순수 간병보험은 보장 범위가 다른 별개 상품 |
