PB01무보장 긴급 상담 전환마이데이터 없음 작동
설명 (필수)
실손·사망보장이 전무하거나 처음 보험 가입을 원하는 사용자를 보닥 플래너에게 연결
발화 키워드 (5개)
보험 하나도 없어요 처음 보험 들려고요 보험 어떻게 시작해요 아무것도 없는데 뭐부터 들어야 해요 보험 없는데 괜찮나요
전환 액션
consult 보닥 플래너와 보험 설계 시작하기
Standalone 답변 (선택)
보험이 전혀 없으시면 지금 당장 실손의료보험부터 준비하시는 것을 권장합니다. 보닥 플래너가 나이·건강 상태에 맞는 보험 설계를 도와드릴 수 있습니다.

PB02암보험 보장 공백 상담 전환마이데이터 없음 작동프롬에이지 없음 CTA
설명 (필수)
암 관련 보장이 부족하거나 암보험 상담 의향을 보이는 사용자를 보닥 플래너에게 연결. 프롬에이지 미연동 시 건강 위험도 기반 진단 유도.
발화 키워드 (6개)
암보험 더 들어야 할까요 암 보장이 부족한 것 같아요 암보험 추천해주세요 암보험 갈아타고 싶어요 암보험 상담 받고 싶어요 암 진단금 얼마면 충분한가요
전환 액션
consult 보닥 플래너와 암보험 점검하기
promage_connect 프롬에이지를 연동하고 내 암 위험도로 진단받기 (프롬에이지 미연동 시 노출)
Standalone 답변 (선택)
암 보장 현황을 분석했습니다. 프롬에이지를 연동하면 내 건강 데이터를 바탕으로 개인 암 위험도까지 반영한 더 정확한 진단이 가능합니다.

PB03심뇌혈관 보장 상담 전환프롬에이지 없음 CTA
설명 (필수)
심혈관·뇌혈관 보장 공백이 감지되거나 관련 보험 상담 의향을 보이는 사용자 연결. 프롬에이지 미연동 시 심뇌혈관 질병위험도 기반 진단 유도.
발화 키워드 (5개)
심혈관 보험 어떻게 들어야 해요 뇌졸중 보장 충분한가요 심장 보험 필요한가요 뇌혈관 보험 추천 심근경색 보장 확인하고 싶어요
전환 액션
consult 보닥 플래너와 심뇌혈관 보장 점검하기
promage_connect 프롬에이지를 연동하고 내 심뇌혈관 위험도로 진단받기 (프롬에이지 미연동 시 노출)
Standalone 답변 (선택)
심뇌혈관 질환은 치료비와 간병비가 크게 발생합니다. 프롬에이지를 연동하면 내 건강 데이터를 바탕으로 심뇌혈관 질병위험도까지 반영한 정밀 진단이 가능합니다.
데이터 미보유 감지
☑ 프롬에이지 없음 → promage_connect CTA 노출
PB04간병보험 상담 전환프롬에이지 없음 CTA
설명 (필수)
노후 간병·치매 보장에 관심을 보이거나 보장 공백이 감지된 사용자를 보닥 플래너에게 연결. 프롬에이지 미연동 시 건강 위험도 기반 간병 필요성 진단 유도.
발화 키워드 (5개)
간병보험 들어야 하나요 치매보험 필요한가요 부모님 간병보험 노후 간병비 어떻게 준비해요 요양보험 궁금해요
전환 액션
consult 보닥 플래너와 노후 돌봄 대비 시작하기
promage_connect 프롬에이지를 연동하고 내 건강 위험도로 간병 필요성 확인하기 (프롬에이지 미연동 시 노출)
Standalone 답변 (선택)
노후 간병비는 공적 장기요양보험만으로는 충당이 어렵습니다. 프롬에이지를 연동하면 내 건강 데이터를 바탕으로 질병위험도까지 반영한 간병보험 필요성 진단이 가능합니다.
데이터 미보유 감지
☑ 프롬에이지 없음 → promage_connect CTA 노출
