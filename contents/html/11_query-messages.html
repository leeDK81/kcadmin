역할: 위험 진단(Concept)이 아니라 "내가 가진 마이데이터를 그대로 보여달라"는 조회형 발화를 감지하는 전용 인덱스. Concept 매칭과 완전히 분리된 인덱스·더 보수적인 기준값을 쓴다(애매하면 위험 진단 우선).
등록 원칙: 여기 등록하는 문장은 "이게 어떤 종류의 조회 요청인지"만 판별하는 용도다. 실제 조회 결과(상품명·금액·날짜 등)는 마이데이터 원본 값을 그대로 바인딩하며 LLM이 재구성하지 않는다.
운영자 등록 화면: mockups_v2/20_query-messages.html에서 실사용 로그를 보며 계속 추가·정리한다. 상세 정책은 context/answer-logic.md "Case 0 — 데이터 조회형" 참조.
QM-01 · 전체 보유 목록 · 17건
QM-01전체 보유 목록17건
정의
사용자가 가입한 보험 상품 전체를 한 번에 나열해달라는 요청
응답 템플릿(안)
"현재 마이데이터에 연동된 보험은 총 N건이에요" + 상품별 리스트(보험명 / 월납입액 · 가입일)
예시 발화 (Q)
내가 가진 보험 다 보여줘 나 지금 가입한 보험 뭐뭐 있어? 가입한 보험 목록 좀 알려줘 내 보험 리스트 뽑아줘 지금 들고 있는 보험 종류 알려줘 나 보험 몇 개나 들었어? 마이데이터 연동한 보험 전부 보여줘 내가 가입한 보험사랑 상품명 다 알려줘 우리 집에 든 보험 몇 개인지 궁금해 내 명의로 된 보험 전체 조회하고 싶어 가입해둔 보험 싹 다 정리해서 보여줘 나 보험 뭐 들었는지 까먹었어, 다 알려줘 전체 가입 내역 확인하고 싶어요 내 보험 현황 한눈에 보고 싶어 보유 중인 보험 상품 목록 알려줘 지금까지 든 보험 다 나열해줘 내 앞으로 되어있는 보험 전부 조회해줘
QM-02 · 특정 상품 가입 여부 · 16건
QM-02특정 상품 가입 여부16건
정의
특정 보험 상품 하나에 가입돼 있는지 확인하는 요청. 상품명은 contents/10_synonym/synonyms.md의 유사어 연결을 거쳐 표준 상품유형으로 정규화된 뒤 마이데이터 조회에 쓰인다.
응답 템플릿(안)
"네, OO에 가입되어 있으시네요" + 해당 상품 상세(보험명 / 월납입액 · 가입일) / "아니요, OO은 확인되지 않아요"
예시 발화 (Q) — 상품명은 유사어 연결 대상이라 표준 상품유형명·유사어가 섞여 있음
나 운전자보험 가입되어 있어? 실손보험 있나? 암보험 들었었나 확인해줘 나 치아보험 가입한 적 있어? 상해보험 들어놨나? 간병보험 있는지 궁금해 화재보험 가입돼 있어? 나 연금보험 든 거 있나? 어린이보험 가입했었나 기억이 안 나 종신보험 들어놨어? 실비보험 있는지 확인 좀 정기보험 가입한 적 있나요? 암보험 가입 여부 알려줘 나 운전자보험 있나 없나 궁금해 태아보험 들었었나? 치과보험 가입돼 있는지 봐줘
QM-03 · 갱신 예정 조회 · 13건
QM-03갱신 예정 조회13건
정의
조만간 갱신 시점이 도래하는 보험이 있는지 확인하는 요청
응답 템플릿(안)
"N개월 내 갱신 예정인 보험이 N건 있어요" + 상품별 상세(보험명 / 갱신 예정일 · 현재 월납입액)
예시 발화 (Q)
나 곧 갱신되는 보험 있어? 갱신 예정인 보험 알려줘 이번 달에 갱신되는 거 있나? 다음 갱신일이 언제야? 보험료 오르는 갱신 시기 다가온 거 있어? 갱신형 보험 중에 곧 만기 되는 거 있나 갱신 앞둔 보험 몇 개나 있어? 재계약 임박한 보험 알려줘 조만간 갱신되는 보험 뭐 있는지 궁금해 갱신 기간 얼마 안 남은 보험 있어? 다가오는 갱신일 알려줘 내 보험 중에 갱신 준비해야 하는 거 있나? 갱신되면 보험료 얼마나 오르는지 미리 알고 싶어
QM-04 · 만료 건수 조회 · 11건
QM-04만료 건수 조회11건
정의
이미 만료됐거나 해지된 보험이 몇 건인지 확인하는 요청
응답 템플릿(안)
"현재 만료된 보험은 N건이에요" + 상품별 상세(보험명 / 가입일 · 만료일)
예시 발화 (Q)
나 만료된 보험은 몇 개야? 끝난 보험 있어? 해지된 보험 몇 건인지 궁금해 만기 지난 보험 알려줘 유지 안 되고 있는 보험 있나? 보장 끝난 보험 몇 개인지 확인하고 싶어 만료돼서 효력 없는 보험 있어? 지금 실효된 보험 몇 개야? 예전에 들었다가 끝난 보험 있나 계약 종료된 보험 몇 건이야? 더 이상 보장 안 되는 보험 몇 개인지 알려줘
확장 원칙: 위 4종은 1차 지원 세트이며 전부가 아니다. 실사용 로그에서 새로운 조회 유형이 발견되면 운영자가 mockups_v2/20_query-messages.html에서 신규 유형을 등록할 수 있다. 신규 유형은 개발팀이 응답 로직을 구현하기 전까지 "개발 반영 대기" 상태다.
