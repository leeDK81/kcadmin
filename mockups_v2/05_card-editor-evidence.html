Evidence 카드

공인 기관 통계를 등록해 Rule 판단 기준값의 근거를 제공합니다. Rule 카드 저장 시 Evidence 1개 이상 연결이 필수이므로 Rule보다 먼저 만드세요.

공인 외부 통계 기반 단일 유형 — 건강보험심사평가원·국민건강보험공단·통계청·금융감독원 등 공인 기관의 통계 수치를 직접 입력합니다.
마이데이터(고객 보험 현황)와 프롬에이지(건강 위험도)는 상담 시 자동으로 제공되므로 Evidence 카드 등록이 불필요합니다.

다음 단계: Evidence 등록 후 → Concept 카드 또는 Rule 카드를 등록하세요.

새 Evidence 카드 임시저장
현재 상태: 임시저장 — 출처 기관·보고서명·지표명·기준값·활용 방법을 입력한 후 [검수 요청]을 클릭하세요.

이 카드는 Rule 카드에서 연결됩니다. 연결은 캔버스(③ 연결)에서 관리됩니다.
