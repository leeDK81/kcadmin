03 연결 정책

카드 간 연결 방향, 캔버스 전용 원칙, 엣지 유형 및 연결 가능 조건

핵심 원칙
카드 연결은 카드 캔버스(00_canvas-main.html)에서만 가능합니다. 각 카드 편집기에서는 연결 상태를 읽기 전용으로 표시하며, "캔버스에서 변경" 버튼으로 캔버스로 이동합니다. 편집기 내 연결 선택 UI(피커·드롭다운·체크박스)는 구현하지 않습니다.
  • 연결 가능 카드 상태: approved(승인완료) + active(라이브)만 허용
  • review(승인요청) 카드: 캔버스에 표시되나 연결 불가
  • draft(임시저장) 카드: 캔버스에 미표시
  • 연결 방향은 CONNECT_RULES를 따름 — 정의되지 않은 방향 연결 불가
  • Playbook 카드는 CONNECT_RULES 적용 안 함 (독립 체인)
CONNECT_RULES — 연결 방향 정의
출발 카드 도착 카드 필수/선택 설명
Concept Risk-type 필수 AI 답변 체인 진입점 형성. Risk-type 연결 필수. 미연결 시 KC 체인 미진입
Risk-type Rule 필수 이 위험 유형에 적용할 판단 Rule 지정. 최소 1개 연결 권장
Rule Evidence 필수 근거 데이터 확보. 최소 1개 연결 필수
Rule Policy 선택 Clark 앱에 표시할 면책 고지 문구를 운영자가 직접 작성. Rule 답변에 고지 문구를 덧붙여야 할 경우 연결
Evidence — — 단말 카드. 출발 연결 없음
Policy — — 단말 카드. 출발 연결 없음
Playbook — — 독립 체인. CONNECT_RULES 미적용. 편집기에서 발화 키워드·CTA 설정
위 방향 외의 연결(예: Evidence → Rule, Policy → Concept)은 시스템에서 차단합니다. UI에서도 허용되지 않은 방향의 연결 버튼은 노출하지 마세요.
엣지(연결선) 유형
준비 연결 (Pending Edge)
점선 파랑 · #1A4A9A · stroke-dasharray:6,4
approved 카드가 1개 이상 포함된 연결. Clark AI 미반영. 라이브 전환 후 active 엣지로 변경.
라이브 연결 (Active Edge)
실선 초록 · #0F6E56
연결된 모든 카드가 active 상태인 연결. Clark AI 즉시 반영.
엣지 상태 전환 조건: 양쪽 카드가 모두 active가 되는 시점(라이브 전환)에 pending → active로 자동 변경됩니다.
카드 상태별 연결 가능 여부
카드 상태 캔버스 표시 연결 가능 엣지 종류 비고
임시저장 ❌ 미표시 ❌ 불가 — 캔버스에 노출되지 않음
승인요청 ✅ 표시 ❌ 불가 — 캔버스에 보이나 연결 버튼 잠김(연결 불가)
승인완료 ✅ 표시 ✅ 가능 준비 연결 (점선 파랑) Clark AI 미반영 상태
라이브 ✅ 표시 ✅ 가능 라이브 연결 (실선 초록) Clark AI 즉시 반영
일시중지 ✅ 표시 △ 조건부 — 기존 연결 유지, 신규 연결 불가
카드 유형별 연결 기준
체인 순서가 아닌 카드 주체 기준으로 정의합니다. 새 카드를 추가하거나 기존 체인 중간에 끼워 넣을 때 이 표를 참고하세요. "유입"은 이 카드가 받을 수 있는 연결, "유출"은 이 카드가 연결할 수 있는 대상입니다.

체인 중간에 카드를 추가할 때: 새 카드의 "유입" 조건을 먼저 충족(앞 카드에서 연결 추가), 그 다음 "유출" 조건 충족(뒤 카드로 연결 추가). 각 카드 유형의 연결 기준만 따르면 되며 체인 순서를 별도로 외울 필요 없음.
연결 추가·제거 액션

연결 추가:

  1. 캔버스 상단 피커에서 포컬 카드 선택
  2. 하단 그리드에서 "연결 가능" 섹션의 카드에 "연결 추가" 버튼 클릭
  3. 확인 모달에서 승인 → 엣지 생성
    • 양쪽 모두 active → active 엣지(실선 초록)
    • 한쪽이라도 approved → pending 엣지(점선 파랑)

연결 제거:

  1. 그리드의 "연결됨" 섹션에서 해당 카드의 "연결 제거" 버튼 클릭
  2. 확인 모달에서 승인 → 엣지 삭제
연결 제거는 Clark AI 반영 카드(active 엣지)에도 즉시 적용됩니다. 라이브 중인 서비스에 영향을 주므로 주의가 필요합니다.
